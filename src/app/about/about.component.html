- ABOUT

Hello, my name is Gamaliel, a passionate introvert who loves building things with code.

My first encounter with code was immediately after highschool when I decided to try and write python scripts that would help me install ported custom roms on my smartphone from XDA (I always wanted the latest and greatest android OS but had an incompatible phone ). After weeks of fails and even bricking my phone, I was able to install my first ported rom. That ignited my passion for code

Fast forward to 2022, i've been working on my skills to solve real world problems and bring ideas to life with code. I recently graduated from Moringa School's 6 month intensive software development bootcamp program.
